# 用户提示词模板

把下面内容复制给 Agent，并上传论文 PDF。

```text
请使用 research-article-report skill 整理这篇论文，输出中文组会式阅读报告。

要求：
1. 先说明文章主要解决什么问题、为什么重要、过去方法有什么不足、作者提出什么路线；
2. 主体用自然连续段落，不要写成大量 bullet points；
3. 方法部分讲清输入输出、数据、模型/实验系统、训练或实验流程、baseline、ablation 和评价指标；
4. 逐图阅读覆盖所有主图，必须精确到 Fig. 1a、Fig. 1b、Fig. 2c-e 这样的子图；
5. 每个子图说明“画什么、读数含义、证明什么、证据链位置”；
6. 单独总结所有计算机完成的部分、核心方法和代码/数据可用性；
7. 单独总结复现需要的关键条件、缺失信息和最容易失败的步骤；
8. 单独总结湿实验内容，并给出大致成本估计；
9. 成本、复现难度或预算若非论文原文披露，必须标注为粗略估计或推断；
10. 保留必要英文术语，第一次出现时用中文简要解释。
```
