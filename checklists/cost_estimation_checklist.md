# 成本估计检查清单

成本估计前检查：

- [ ] 论文是否披露真实成本？如果没有，必须写“粗略估计”。
- [ ] 是否只估计文章中确实做过、或复现关键结论必需的实验？
- [ ] 是否区分小型复现、中等项目、接近原文规模？
- [ ] 是否说明成本受地区、平台、通量、样品难度、失败率和外包比例影响？
- [ ] 是否避免给出过于精确的单一价格？
- [ ] 是否对高成本实验单独说明，例如大规模文库筛选、cryo-EM、X-ray、动物实验、商业靶蛋白采购？
- [ ] 是否说明计算成本和湿实验成本的边界，例如 GPU 推理、结构预测、数据处理是否计入？
- [ ] 是否明确哪些价格是经验估计，而非论文原始数据？
