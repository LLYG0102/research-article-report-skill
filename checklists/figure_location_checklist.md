# 子图定位检查清单

逐图写作时检查：

- [ ] 是否建立了 Fig. 1 到最后一张主图的清单？
- [ ] 是否写到了每个主图的关键子图，例如 `Fig. 1a`、`Fig. 1b`、`Fig. 1c`？
- [ ] 是否将流程图、模型图、结构图、显微图、统计图、功能曲线分开解释？
- [ ] 是否说明坐标轴、颜色、曲线、结构标注、比例尺或读出指标的含义？
- [ ] 是否把子图与正文结论关联？
- [ ] 是否指出控制组、阳性/阴性对照、baseline 或 benchmark？
- [ ] 是否解释关键指标，如 EC50、IC50、Kd、AUC、RMSD、pAE、pLDDT、FDR、pA2、Emax？
- [ ] 是否区分 prediction、simulation、design model、experimental structure 和 functional assay？
- [ ] 如果图像不清晰或无法渲染，是否明确说明需要原图确认？
