# 阅读报告质量检查清单

生成报告前后检查：

- [ ] 是否说明文章要解决的核心科学问题，而不是只翻译摘要？
- [ ] 是否讲清楚过去方法为什么不够，以及本文方法改变了什么？
- [ ] 是否准确描述输入、输出、数据、模型/实验系统、评价指标和验证链条？
- [ ] 是否覆盖所有主图，并覆盖关键 Extended Data/Supplementary 图？
- [ ] 图像介绍是否精确到 `Fig. 1a`、`Fig. 1b`、`Fig. 2c-e`？
- [ ] 每个子图是否说明“画什么、读数含义、证明什么、证据链位置”？
- [ ] 是否保留关键数字，例如样本量、成功率、Kd、IC50、RMSD、AUC、pAE、分辨率？
- [ ] 是否有单独章节总结所有计算部分？
- [ ] 是否有单独章节总结复现条件和缺失信息？
- [ ] 是否有单独章节总结湿实验？
- [ ] 成本估计是否明确标注为估算，并给出范围而非精确单价？
- [ ] 是否区分论文事实、作者结论、你的推断和复现建议？
- [ ] 是否避免把 binding 直接等同于 function？
- [ ] 是否避免把预测结构说成实验结构？
- [ ] 是否避免把 in silico success 写成 wet-lab success？
- [ ] 是否在最后写出具体启发、可迁移思路和不能外推的边界？
